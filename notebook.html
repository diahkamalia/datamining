Statistik Deskriptif#

Statistik Deskriptif adalah proses mengkategorikan dan menggambarkan informasi. Statistik inferensial, di sisi lain, mencakup proses menganalisis sampel data dan menggunakannya untuk menarik kesimpulan tentang populasi dari mana data itu diambil.

Statistik Deskriptif dapat berguna untuk dua tujuan:

  1. Untuk memberikan informasi dasar tentang variabel dalam kumpulan data

  2. Untuk menyoroti hubungan potensial antar variabel. Tiga statistik deskriptif yang paling umum dapat ditampilkan secara grafis atau bergambar dan merupakan ukuran dari: Metode Grafis/Bergambar.

Metode Statistik Deskriptif:

  • Pemusatan data pada data tunggal dan data berkelompok (mean, median, dan modus)

  • Ukuran letak data tunggal dan data berkelompok (kuartil dan desil)

  • Ukuran penyebaran data pada data tunggal (jangkauan, jangkauan antar kuartil, simpangan, simpangan rata-rata)

  • Ukuran penyebaran data pada data berkelompok (jangkauan, jangkauan antar kuartil, simpangan kuartil, simpangan baku, dan ragam)

Mean#

Mean atau rata-rata adalah perbandingan jumlah seluruh data dan banyaknya data.

Formula :

Data Tunggal
Data tunggal adalah menjumlahkan semua nilai data lalu membaginya dengan banyaknya data.

\[ \bar{x}=\frac{x_1+x_2+x_3+\ldots+x_n}{n} \]

Dengan :
\(\bar{x}=\text { Rata }-\text { rata }\)
\(x_1+x_2+x_3+\ldots+x_n=\) Nilai data
\(n=\) Banyaknya data

Data Berkelompok
Jumlah seluruh nilai data adalah jumlah frekuensi kali nilai data, dan banyaknya data diwakili oleh jumlah frekuensi keseluruhan

\[ \bar{x}=\frac{\sum f_i x_i}{n} \]

Dengan : $\( \begin{aligned} \bar{x} &=\text { Rata }-\text { rata } \\ f_i &=\text { Frekuensi } \\ x_i &=\text { Nilai tengah } \\ n &=\text { Total frekuensi atau data } \end{aligned} \)$

#mean

Median#

Median adalah nilai tengah dari data yang ada. Gue pribadi punya cara sendiri untuk ngapalin apa itu median. Dari tulisannya kan awalnya ada ‘med’, seperti medium. Nah, medium itu biasanya tengah-tengah, di bawah large tapi di atas small. Jadi, gue selalu inget banget median itu artinya nilai tengah.

Formula :

Data Ganjil

Jumlah data ganjil : $\( M_e=\frac{X_{(n-1)}}{2} \)\( Jumlah data genap: \)\( M_e=\frac{X_{\left(\frac{n}{2}\right)}+X_{\left(\frac{n}{2}+1\right)}}{2} \)$

Data Berkelompok
Rumus menghitung median untuk data berkelompok sama dengan rumus untuk mencari kuartil kedua pada data berkelompok. Median sama saja dengan kuartil kedua atau \(Q _2\)

\[ M_e=L+c\left(\frac{\frac{1}{2} n-f_k}{\sum f_i}\right) \]

mid = {'xmin':c['xmin'].median(),

Modus#

Modus adalah nilai yang frekuensi kemunculannya paling banyak. Dengan kata lain, modus merupakan nilai yang paling sering muncul dibandingkan dengan nilai-nilai lainnya.

Formula :

Data Tunggal

Contoh :
Tentukan modus dari data di bawah ini.
2, 1, 4, 1, 1, 5, 7, 8, 9, 5, 5, 10
Jawab:
Data yang sering muncul adalah 1 dan 5. Jadi modusnya adalah 1 dan 5

Data Berkelompok

$\( M_o=L+c\left(\frac{d_1}{d_1+d_2}\right) c \)\(<br> Dengan : <br> \)Mo\( = Modus<br> \)L\( = Tepi bawah kelas yang memiliki frekuensi tertinggi (kelas modus) i = Interval kelas<br> \)d _1\( = Frekuensi kelas modus dikurangi frekuensi kelas interval terdekat sebelumnya<br> \)d _2$ = frekuensi kelas modus dikurangi frekuensi kelas interval terdekat sesudahnya

Skewness / Kemiringan#

Derajat distorsi dari kurva lonceng simetris atau distribusi normal. Ini mengukur kurangnya simetri dalam distribusi data Untuk menghitung derajat distorisi dapat menggunakan Koefisien Kemencengan Pearson yang diperoleh dengan menggunakan nilai selisih rata-rata dengan modus dibagi simpangan baku.

Formula :

\[ \operatorname{Kurt}[X]=\mathrm{E}\left[\left(\frac{X-\mu}{\sigma}\right)^4\right]=\frac{\mathrm{E}\left[(X-\mu)^4\right]}{\left(\mathrm{E}\left[(X-\mu)^2\right]\right)^2}=\frac{\mu_4}{\sigma^4}\]
